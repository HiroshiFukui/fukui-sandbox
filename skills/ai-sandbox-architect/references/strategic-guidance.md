# Strategic Guidance

Analyze the current state of the AI Sandbox module and guide the owner toward the most logical next action.

## What Success Looks Like

The owner understands:
- Which sandboxes are currently active, planned, or in need of maintenance.
- Which workflows are appropriate to run next (e.g., "The blueprint for 'my-agent' is ready. Should we run the `create-ai-sandbox` workflow?").
- Any potential conflicts or optimization opportunities identified in the shared memory.

## Memory Integration

Read `{project-root}/_bmad/memory/sa/index.md` on activation. Scan for missing validation reports or outdated blueprints.

## After the Session

Update the status of any sandbox in `index.md` if the conversation led to a change in state or intent.
